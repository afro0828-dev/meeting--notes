2

論点

OB会計と学生援助
  • • OB会計残高は約330万円（繰越＋年会費収入）
  • • 毎年35万円前後を学生援助に充当（道具・勧誘費など）
  • • 赤字は避けつつ、積み上がりすぎないバランスが必要
  • • OB納付率は約3分の1（自動振込と振込用紙の併用）
学生保険と安全管理
  • • プレーヤー全員の保険加入・更新の徹底
  • • 保険料はOB会からの援助も視野に入れる
  • • 手続き漏れリスク（更新忘れ）への対策
  • • マネージャー任せにせず、主務・OBがチェック
名簿・情報管理とデジタルツール
  • • 名簿はUSB紛失をきっかけにクラウド管理へ移行中
  • • EvernoteからNotionへの移行を進行
  • • 共有しやすいスペース（Notion / Slack 等）の整備案
  • • 紙の郵送費を減らし、デジタル連絡へ移行したい
勧誘戦略とイベント
  • • 現状は人数不足（単独チームは長期計画）
  • • 他部はクルージング等の派手なイベントで勧誘
  • • ラグビー部も「まず存在を知ってもらう場」が必要
  • • ご飯・イベント・雰囲気体験など、学生目線の企画が重要
  • • OBができるのは「お金」と「場の提供」が中心
農会・赤ジャージ授与の日程設計
  • • これまで: 1月第4土曜前後で開催
  • • 6年生の国家試験・各学年試験とのバッティング問題
  • • 2〜3月・5月など他候補も議論
  • • 「赤ジャージを手渡しする場」としての価値維持
60周年記念パーティー
  • • 目的・ゴール（何を得たいか）が不明瞭
  • • 実行委員の負担が非常に大きい（学会主催並み）
  • • 予算300万円を「学生支援」か「記念行事」かで迷い
  • • 同窓会代行・学会代行のような外部委託案もあり
3

決定事項

OB会計と学生援助
  • ✓ OB会計残高は約330万円を維持しつつ、赤字は避ける
  • ✓ 学生援助は原則35万円前後＋個別要望に応じて追加
  • ✓ 資金の主目的は「学生支援」であり、OBの遊興ではない
学生保険・安全
  • ✓ プレーヤー全員が保険に加入する方針
  • ✓ 保険料は必要に応じてOB会から支出してよい
  • ✓ 更新漏れを防ぐため、主務＋OBでダブルチェックする
学生要望の窓口体制
  • ✓ 学生からの要望（ジャージ・タックルバッグ等）は窓口OBが一括受付
  • ✓ 要望内容はLINEで共有し、必要額をOB会から振込
デジタル化の方向性
  • ✓ 名簿等の重要データはクラウド管理へ移行する
  • ✓ Notion等で、後輩に引き継ぎやすい情報基盤を作る方針
農会・赤ジャージ授与
  • ✓ シーズン終了（12月）後〜国家試験前後の時期に開催する方針
  • ✓ 6年生が参加しやすい日程を最優先する
  • ✓ 暫定案として「2025年2月第3土曜 16時開始」で検討を進める
勧誘に対するOBのスタンス
  • ✓ 勧誘の現場は学生主体とし、OBは資金と後方支援に集中
  • ✓ 勧誘イベントに必要な費用は、計画があればOB会から支出可
4

保留事項

学生保険のプランと運用

どの保険商品を採用し、誰が更新管理を担うかは要検討（学生＋主務＋OB）

デジタル情報基盤の設計

Notion / Slack 等でどのようなワークスペースを作るか（管理者・公開範囲・引き継ぎルール）

勧誘イベントの中身

屋形船・食事会・練習見学会など、どの施策が学生に響くかは学生側で要検討

農会の日程・形式

国家試験・各学年試験・卒業旅行との兼ね合いを踏まえた最適解は、学生と再度擦り合わせが必要

60周年記念パーティー
  • • 実施するかどうか（やらない選択肢も含めて）
  • • 実施する場合の目的（新規OB参加促進か、単なる同窓会か）
  • • 実行委員長・実務担当を誰が担うか
  • • 外部業者（同窓会代行・学会代行等）への委託可否
OB会費徴収・決済手段

口座振替・クレジットカード・PayPay等をどう組み合わせるかは今後の検討課題
